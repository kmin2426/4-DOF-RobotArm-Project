This repository contains the final project for the course  
**Microprocessor Application Systems**,  
taught by **Prof. Soon-Min Hwang**

## Project Overview
The goal of this project is to develop a **4-DOF robotic arm system** capable of
precise servo motor control using a microcontroller.

The system integrates multiple hardware modules and demonstrates:
- Multi-axis servo control
- PWM-based actuation
- Peripheral device integration
- Embedded software development

---

## Hardware Components
- **Servo Motors**: MG996R, DS3120MG  
- **Servo Driver**: PCA9685  
- **Microcontroller**: Arduino (ATmega328P)  
- **Bluetooth Module**: HC-06  
- **Speaker Module**: ISD1820  
- **Power Supply**: External power for servo motors  

---

## System Architecture
- The Arduino (ATmega328P) controls all actuators and peripherals
- PCA9685 is used to generate PWM signals for stable multi-servo control
- Bluetooth (HC-06) enables wireless communication
- Speaker module provides audio feedback
- A 3D-printed mechanical structure forms the robotic arm body

---

## Key Features
- 4-axis robotic arm control
- PWM-based servo motor actuation
- Bluetooth-based wireless command input
- Modular embedded software design
- Integration of mechanical, electrical, and software components
